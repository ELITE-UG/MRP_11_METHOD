# MRP (MATERIAL REQUIREMENT PLANNING) WITH 11 METHOD
The application begins by configuring the Streamlit page layout and implementing custom CSS to create a modern and user-friendly interface. The interface includes a responsive dashboard, customized sidebar, interactive buttons, and a clear application title, "MRP Lot Sizing Calculator," providing users with an intuitive environment for inventory planning.
Lot Sizing Glossary
built-in glossary provides concise explanations of the eleven lot sizing methods implemented in the system, including:
Lot-for-Lot (L4L)
Economic Order Quantity (EOQ)
Period Order Quantity (POQ)
Fixed Order Quantity (FOQ)
Fixed Period Requirement (FPR)
Incremental Unit Cost (IUC)
Least Total Cost (LTC)
Least Unit Cost (LUC)
Part Period Balancing (PPB)
Silver-Meal (SM)
Wagner-Whitin (WW)
